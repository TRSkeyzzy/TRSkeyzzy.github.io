GitHubの使い方


【最初に】githubとは

GitHub（ギットハブ）はソフトウェア開発プロジェクトのための共有ウェブサービスであり、Gitバージョン管理システムを使用する。 Ruby on RailsおよびErlangで記述されており、GitHub社によって保守されている。 主な開発者はChris Wanstrath、PJ Hyett、Tom Preston-Wernerである。 GitHub商用プランおよびオープンソースプロジェクト向けの無料アカウントを提供している。 2009年のユーザー調査によると、GitHubは最もポピュラーなGitホスティングサイトとなった。


【コマンド紹介】

ファイルの作成 git add hello.html
変更結果をローカルリポジトリにコミットする（git commit） git commit -m "add new file"
ファイルが追加されているか確認 git status
ローカルリポジトリをプッシュしてリモートリポジトリへ反映させる git push origin master
ファイルやディレクトリをインデックスに追加 git add [file_pattern]
ブランチに対して各種操作を行う! git branch
ローカルリポジトリのブランチを切り替える git checkout [branch-name]
ローカルリポジトリのコミット履歴を閲覧 git log –n 10
既存のリモートリポジトリをローカルに落とす git clone [url]
